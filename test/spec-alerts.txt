
# Alerts (GitHub-style admonitions)

## Basic NOTE alert

```````````````````````````````` example
> [!NOTE]
> This is a note
.
<blockquote class="alert alert-note">
<p>This is a note</p>
</blockquote>
````````````````````````````````

## Basic WARNING alert

```````````````````````````````` example

</blockquote>

## Alert with inline formatting

```````````````````````````````` example
> [!NOTE]
> This **is** a *formatted* message
.
<blockquote class="alert alert-note">
<p>This <strong>is</strong> a <em>formatted</em> message</p>
</blockquote>
````````````````````````````````

## Alert with multiple paragraphs

```````````````````````````````` example
> [!NOTE]
> First paragraph
>
> Second paragraph
.
<blockquote class="alert alert-note">
<p>First paragraph</p>
<p>Second paragraph</p>
</blockquote>
````````````````````````````````

## Empty alert (just the type line)

```````````````````````````````` example
> [!NOTE]
.
<blockquote class="alert alert-note">
</blockquote>
````````````````````````````````

## Not an alert: [!TYPE] not on first line

```````````````````````````````` example
> Some text
> [!NOTE]
> More text
.
<blockquote>
<p>Some text
[!NOTE]
More text</p>
</blockquote>
````````````````````````````````

## Not an alert: text after [!TYPE] on same line

```````````````````````````````` example
> [!NOTE] extra text
> Content
.
<blockquote>
<p>[!NOTE] extra text
Content</p>
</blockquote>
````````````````````````````````

## Nested blockquote inside alert

```````````````````````````````` example
> [!NOTE]
> Content
>
> > Nested quote
.
<blockquote class="alert alert-note">
<p>Content</p>
<blockquote>
<p>Nested quote</p>
</blockquote>
</blockquote>
````````````````````````````````

## Alert with list content

```````````````````````````````` example
> [!TIP]

<li>Item 2</li>

## Alert with code block

```````````````````````````````` example
> [!NOTE]
> Example:
>
> ```js
> console.log("hello")
> ```
.
<blockquote class="alert alert-note">
<p>Example:</p>
<pre><code class="language-js">console.log(&quot;hello&quot;)
</code></pre>
</blockquote>
````````````````````````````````

## Alert with link

```````````````````````````````` example
> [!NOTE]
> See [example](http://example.com) for details
.
<blockquote class="alert alert-note">
<p>See <a href="http://example.com">example</a> for details</p>
</blockquote>
````````````````````````````````
